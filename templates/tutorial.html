從 Access 匯出資料

快速匯出

如果並沒有設定好的匯出選項，可以參考下方的完整匯出教學。

1. 開啟 Access 應用程式，確認資料庫中有 comm_list、comm_name、teacher 三張表格。選擇上方工具列「External Data」→「Saved Exports」。

2. 跳出一個「Manage Data Tasks」視窗。第一次操作時，可以把三個表格對應的路徑改成程式資料夾 ...\raw_export\表格名稱.txt。

3. 完成路徑設定後，只要依序選取三個表格，點擊左下角「Run」的按鈕即可。

4. 到首頁選擇「重新載入資料庫」即可。完成會看到一個「Successfully reloaded 字樣」。

完整匯出教學

如果能用前述的方法匯出，則可不用閱讀。

1. 開啟 Access 應用程式，確認資料庫中有 comm_list、comm_name、teacher 三張表格。

2. 按順序在右側 Tables 中三張表格按右鍵，選擇「Export」→「Text File」。

3. 此時會出現「Export Text Wizard」視窗。本頁面維持預設 Delimited 設定。按下一步。

4. 選擇輸出的位置，請記得保持原本 表格名稱.txt 的格式。

5. 選擇輸出的格式細節，請照截圖的選項或預設值設定。完成之後按下左下角「Advanced...」按鈕。

6. 會跳出新視窗。將 code page 改成「Unicode (UTF-8)」。

7. 到這邊已經完成所有設定，請繼續按下一步直到 Wizard 視窗關閉。確認系統中剛剛設定的路徑已有輸出檔案。

8. 重覆上述動作完成三個表格。最後將三個 txt 檔案複製至程式目錄的 raw_export\ 資料夾裡。
